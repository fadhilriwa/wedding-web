FOLDER MUSIK UNDANGAN
=====================

Letakkan file musik (.mp3 / .ogg) di folder ini.

Lalu di Admin Dashboard → Pengaturan → URL Musik, isi dengan:
  music/nama-file.mp3

Contoh:
  music/romantis.mp3
  music/wedding-song.mp3

Tips:
- Format MP3 didukung semua browser
- Ukuran file ideal: di bawah 5 MB agar cepat loading
- Gunakan format .webm atau .ogg sebagai backup untuk Firefox lama
